* ЗАДАНИЕ 
Задание в группах: Доделать калькулятор с функциями вычислений рациональных и комплексных чисел, 
должно вестись логирование, должна быть разработанная архитектура ПО, наличие отдельных модулей. 
Всё выложить на гитхаб со скриншотами успешной работы приложения во всех реализованных режимах.

-----

Работа велась 1-м человеком, т.к. сильно отстал по учёбе.
Для работы необходимы модули numpy и getch.
Для установки numpy введите в консоли:
"pip install numpy" или "pip3 install numpy" для Mac.
Для установки getch введите в консоли:
"pip install py-getch" или "pip3 install py-getch" для Mac.

Numpy используется для округления комплексных чисел.
Getch используется для приостановки программы на паузу. Пауза необходима, чтобы экран не обновлялся, т.к. далее идёт очистка экрана.

Очистка экрана производится с помощью стандартного модуля os.

Логгирование использует запись даты и времени с помощью стандартного модуля datetime.
